About US

NoRecoil started on the 6th of July 2022 (My Birthday), and is set out to be a community-driven team that will help code useful software and scripts for junior developers and the general public, most of our source code will be released via GitHub as the centralized platform for open-source code and projects, I started this 'team' as a test of my abilities for building communities, healthy and sustainable communities.

This is my first real, fully motivated project and i intend on populating it with many intelligent minds and great people, I would like to thank the people that have joined NoRecoil on personal or voluntary reasons.
